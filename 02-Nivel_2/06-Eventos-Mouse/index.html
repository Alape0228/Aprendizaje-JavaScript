Evento: Click

Evento generado por un solo click, el evento se aplica indiferentemente que el click se de a su caja padre o cada uno de sus elementos o cajas internas

Evento: dblclick

Evento generado al usar doble click no lento o separado

Evento: mouseover

* Ocurre cuando el puntero se mueve sobre un elemento o sobre uno de sus hijos

* Si el puntero del mouse entra a otro div hijo del elemento se aplicara el evento o si entra a algun boton u otro elemento hijo se seguira aplicando el evento repetitivamente hasta que se salga del contenedor padre como tal que es el que contiene el evento

Evento: mouseout

* Ocurre cuando se mueve el puntero fuera de un elemento o de sus elementos secundarios

* El evento se aplica a la caja contenedora si el mouse se para dentro de su caja no generara efecto pero si sale de la caja si

* Si el mouse esta dentro de uno de los elementos hijos y el puntero intenta salir de sus elementos hijos se aplicara el efecto

Evento: contextmenu

*Ocurre con un click en el boton derecho en un elemento para abrir un menu contextual

Evento: mouseenter

* Ocurre cuando el puntero se mueve sobre un elemento

* Si el mouse entra al elemento se aplica el efecto y mientras este pasando el mouse por el resto de sus elementos hijos no se aplicara el efecto si no cuando el puntero vuelva a salir y vuelva a regresar al elemento

se recomienda usar en internet explorer

Evento: mousemove

* Ocurre cuando el puntero se mueve mientras esta sobre un elemento

Evento: mouseleave

* Ocurre cuando el puntero se mueve fuera de un elemento

Evento: mousedown

* Ocurre cuando el puntero se mueve mientras esta sobre un elemento

Evento: mouseup

* Ocurre cuando un usuario suelta un boton del mouse sobre un elemento
